## API

### Avatar

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| children | ReactNode | 无 | 内容，文字或react组件 |
| shape | circle \| square | circle | 形状 |
| src | string | 无 | 图片地址 |
| bordered | boolean | false | 是否显示边框 |
| icon | FontAwesome | 无 | 图标名称 |
| size | small \| large \| default \| number | default | 大小 |
